# Настройка Telegram Login Widget

## Шаг 1: Привязка домена к боту в BotFather

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду: `/setdomain`
3. Выберите вашего бота из списка (например, `PrePromo_bot`)
4. Для **локального тестирования** введите: `localhost:8080`
5. Для **production** введите ваш домен (например: `ваш-домен.ru`)

**Важно:** 
- Для локального тестирования используйте `localhost:8080` (без http://)
- Для production используйте домен без протокола (например: `docugen.ru`)

## Шаг 2: Проверка настройки

После выполнения `/setdomain` BotFather должен ответить:
```
Success! Domain example.com is now linked to your bot.
```

## Шаг 3: Проверка виджета

Виджет должен автоматически загрузиться на странице авторизации.

Если виджет не появляется:
1. Откройте консоль браузера (F12)
2. Проверьте ошибки в консоли
3. Убедитесь, что домен добавлен в BotFather
4. Перезагрузите страницу

## Текущие настройки виджета

В коде уже настроено:
- `data-telegram-login="PrePromo_bot"` (из VITE_TELEGRAM_BOT_USERNAME)
- `data-size="large"`
- `data-onauth="handleTelegramAuth"`
- `data-request-access="write"`

## Что делать сейчас

1. **Откройте BotFather в Telegram**
2. **Отправьте:** `/setdomain`
3. **Выберите бота:** `PrePromo_bot`
4. **Введите домен:** `localhost:8080` (для локального тестирования)
5. **Перезагрузите страницу** http://localhost:8080
6. **Проверьте виджет** - должна появиться кнопка "Login with Telegram"

После этого виджет должен работать и открывать окно Telegram для авторизации.
